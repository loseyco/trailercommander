# AI Agent Rules & Guidelines

1. **Test Before You Ask**: ALWAYS use the `browser_subagent` to test new frontend features, routing logic, or UI components. Confirm that they work visually and functionally before asking the user to review them. Never assume code works just because it compiled.
2. **Offline-First Resilience**: All architecture decisions must respect the "Trailer Off-Grid" requirement. Local network capabilities must always take precedence over cloud capabilities.
3. **Vanilla CSS Standard**: Maintain premium aesthetics (glassmorphism, modern typography) using Vanilla CSS. Do not use Tailwind or external CSS frameworks unless explicitly directed.
